# Project procedures

`dev/procedures/` is where CrosswordChat's project-specific additional instructions
reside — procedures and conventions that apply to this repository only, layered on top
of the shared Claudinite canon synced into `.claudinite/`.

This file is hard-imported from the root `CLAUDE.md`, so it loads into every session.
Add new instruction files to this folder and `@`-import them from here so they load too.
